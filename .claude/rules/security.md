# 安全规则

## 敏感信息

- **禁止**在代码中硬编码:
  - API Token / Secret Key
  - 用户密码
  - 数据库连接字符串
  - 私钥/证书
- 所有敏感配置通过环境变量注入
- `.env` 文件**不提交**到版本控制

## 权限检查

- 前端权限检查是用户体验层面的，最终权限由后端强制
- 敏感操作前仍需后端验证，前端检查仅用于隐藏/禁用 UI
- 使用 `hasPermission()` 而非自行判断

## XSS 防护

- React 默认转义用户输入
- **禁止**使用 `dangerouslySetInnerHTML`，除非内容完全可信且经过 sanitization
- 用户生成的文本始终作为纯文本渲染

## CSRF

- API 请求使用 Axios 实例，自动处理 CSRF Token
- 不要自行在请求中添加不信任的 headers

## 依赖安全

- 定期运行 `pnpm audit` 检查漏洞
- 更新依赖时关注安全公告
- 避免使用已知漏洞版本
